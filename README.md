# FUTURE_DS_01 - Business Sales Performance Analytics

## Project Overview

This Power BI dashboard analyzes business sales performance across products, regions, and customer segments. The dashboard helps identify revenue trends, top-performing products, and sales opportunities.

## Objectives

* Monitor overall sales performance
* Identify top-selling products
* Analyze revenue trends
* Evaluate regional performance

## Tools Used

* Power BI
* Excel
* Power Query
* DAX

## Key KPIs

* Total Revenue
* Total Orders
* Average Order Value
* Profit Margin
* Top Performing Products

## Dashboard Features

* KPI Cards
* Revenue Trend Analysis
* Product Performance Analysis
* Regional Sales Analysis
* Interactive Filters and Slicers

## Key Insights

* A small group of products contributes significantly to total revenue.
* Certain regions consistently outperform others.
* Sales trends reveal seasonal demand patterns.
